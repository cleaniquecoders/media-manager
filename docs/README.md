# Documentation

Welcome to the Media Manager documentation. This guide will help you integrate and use the
Media Manager package in your Laravel applications.

## Overview

Media Manager is a Laravel package that provides a complete media management solution built on
top of [Spatie Media Library](https://spatie.be/docs/laravel-medialibrary). It offers a modern,
user-friendly interface for uploading, browsing, organizing, and managing media files.
