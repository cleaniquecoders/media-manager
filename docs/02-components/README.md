# Components

This section documents all Livewire components, traits, and services provided by Media Manager.

## Table of Contents

### [1. Media Browser](01-media-browser.md)

Full-featured media browsing interface with filtering, searching, and bulk operations.

### [2. Media Uploader](02-media-uploader.md)

Upload component for adding files to a specific model and collection.

### [3. Media Collection](03-media-collection.md)

Inline collection editor with drag-and-drop reordering.

### [4. Media Picker](04-media-picker.md)

Modal component for browsing and selecting existing media.

### [5. Media Service](05-media-service.md)

The core service class and facade for programmatic media management.

### [6. Traits](06-traits.md)

Available traits and concerns for extending your models and components.

## Component Overview

| Component | Purpose | Use Case |
|-----------|---------|----------|
| `media-browser` | Browse all media | Admin media library page |
| `media-uploader` | Upload to model | Form with file upload |
| `media-collection` | Edit collection inline | Edit page with gallery |
| `media-picker` | Select existing media | Link existing media to model |
